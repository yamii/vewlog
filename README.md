A log viewer tool for different projects using golang.

## Prerequisites
Run the following commands in your command line
```
go get -u github.com/gin-gonic/gin
```

## Development
* Clone this repo
* Install Go Live Reloading `go get github.com/oxequa/realize`
* run `realize start`
